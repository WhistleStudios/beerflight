Beer Flight

Introducing a new way to quickly present web design prototypes

Beer Flight is a tool to help demonstrate different variations of a web design prototype. It's perfect for UX designers familiar with HTML & CSS to show (not tell) people what an experience would be like.

Get started

  1. Add Beer Flight to your HTML prototype: 
    <script src="/path/to/beerflight.js"></script>
  2. Add your taster buttons, CSS selectors and classes: 
    <button beerflight-taster="Light" beerflight-selector="#header" beerflight-class="try-light"></button>
<button beerflight-taster="Medium" beerflight-selector="#header" beerflight-class="try-medium"></button>
...
  3. Pour and serve
